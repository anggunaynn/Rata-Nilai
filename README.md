# Rata-Nilai
Untuk menghitung rata-rata nilai beberapa siswa berdasarkan nilai yang dimasukkan oleh pengguna
Mengotomatisasi perhitungan rata-rata nilai siswa agar lebih cepat, akurat, dan tidak perlu dihitung secara manual.
